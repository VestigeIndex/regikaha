# 4. Modelo de negocio

## Clientes

El cliente usa Regi Kaha gratis.

Puede:

- buscar profesionales
- comparar perfiles
- ver portfolio
- consultar precios orientativos
- pedir presupuesto
- contactar con profesionales
- leer reseñas
- valorar al finalizar el servicio

## Profesionales

El profesional paga una cuota fija.

### Precio

**19,95 €/mes + IVA/VAT** en Autónomo Nacional y **49,95 €/mes + IVA/VAT** en Multi-mercado Pro

o

**215,46 €/año + IVA/VAT** o **539,46 €/año + IVA/VAT** con 10% de descuento anual

### Oferta fundadora

**Primeros 300 profesionales verificados: 5 meses gratis.**

Después pueden continuar con la cuota normal.

## Reglas de monetización

No hay:

- ranking comprado
- pago por aparecer primero
- publicidad interna pay-to-win
- comisión sobre el valor del proyecto
- precio oculto por mensaje
- créditos artificiales
- trampas de visibilidad

Todos los profesionales compiten por mérito.

# 19. Infraestructura inicial

Construir pensando en una fase inicial low-cost:

- Cloudflare Pages
- Workers Free
- D1 Free
